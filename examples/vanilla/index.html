How to Use

  1. Enter original and modified content in the textareas above
  2. Click "Compare Documents" to see the diff
  3. Use "Accept All" to accept all changes
  4. Use "Reject All" to reject all changes
  5. The plugin provides three callbacks:
    • onmerge(action, origincontent, modifiedContent, resultContent) - Called when accepting or rejecting changes
    • oncomplete() - Called when all changes are processed
    • onDiffStateChanged(currentIndex, count) - Called when diff state changes
